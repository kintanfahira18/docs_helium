# Scan Ulang
![Scans Capture](/img/capture/direct-scan.png)
:::note Catatan

Untuk mengaktifkan tombol *Rescan*, pengguna terlebih dahulu klik *checkbox* target.

:::

Untuk melakukan *scan* ulang pada target, pengguna dapat klik tombol *Rescan*, maka akan tampil pengisian seperti halnya pada proses *scanning*. Pengguna dapat mengisi sesuai dengan *VAPT tools* yang digunakan, misalnya *Website Scanner* berikut.
![Rescan Capture](/img/capture/rescan.png)
Pengguna dapat mengikuti langkah-langkah berikut.
1. Inputkan nama task pada textbox Task Name 
2. Inputkan path pada textbox Path (opsional)
3. Klik Enabled untuk mengaktifkan Multiple Scan (opsional)
4. Pilih salah satu scan option yaitu antara Full Scan atau Basic Scan
5. Klik Enabled untuk mengaktifkan Authentication (opsional)
6. Klik Enabled untuk mengaktifkan Schedule Scan (opsional)
7. Klik checkbox agree Terms of Service
8. Klik tombol Start Scan untuk memulai scan, klik Cancel jika ingin membatalkan
