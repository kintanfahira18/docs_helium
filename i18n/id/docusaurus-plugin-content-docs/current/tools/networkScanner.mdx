# Network Scanner
Menemukan layanan jaringan yang ketinggalan jaman, konfigurasi perangkat lunak yang tidak aman, patch keamanan yang hilang, dan kerentanan lainnya.

### Akses *Network Scanner* melalui Menu Navigasi
:::note Catatan

Pengguna dapat melakukan pemindaian menggunakan VAPT Tools melalui menu navigasi "VAPT Tools". Dengan mengklik menu "VAPT Tools", akan muncul berbagai macam alat, kemudian klik alat *Network Scanner*.

:::
![Select Network Scanner Navbar Capture](/img/capture/vapt_navbar/Select-network-scanner-navbar.png)

![Network Scanner Capture](/img/capture/vapt_navbar/network-scanner.png)

Untuk *scanning* menggunakan *Network Scanner*, pengguna dapat mengikuti langkah-langkah berikut.
1. Inputkan nama *task* pada *textbox Task Name* 
2. Inputkan nilai target berupa *domain/URL/IPs*
3. Pilih salah satu *scan option* yaitu antara *Full Scan* atau *Basic Scan*
4. Klik *Scheduled* untuk *Enabled* atau aktifkan *Schedule Scan* (opsional)
5. Klik *Multiples Scans* untuk *Enabled* atau aktifkan *Multiple Scans* (opsional)
6. Klik *checkbox agree Terms of Service*
7. Klik tombol ***Apply*** untuk memulai *scan*

### Akses *Network Scanner* melalui Halaman Targets
:::note Catatan

Target yang dipilih akan dipindai dengan alat yang tersedia. Pilih Alat *Network Scanner*.

:::
![Select Network Scanner Capture](/img/capture/vapt_target/Select-network-scanner.png)

![Network Scanner Capture](/img/capture/vapt_target/network-scanner.png)

Untuk *scanning* menggunakan *Network Scanner*, pengguna dapat mengikuti langkah-langkah berikut.
1. Inputkan nama *task* pada *textbox Task Name*  
2. Klik *Enabled* untuk mengaktifkan *Multiple Scan* (opsional)
3. Pilih salah satu *scan option* yaitu antara *Full Scan* atau *Basic Scan*
4. Klik *Enabled* untuk mengaktifkan *Schedule Scan* (opsional)
5. Klik *checkbox agree Terms of Service*
6. Klik tombol *Start Scan* untuk memulai *scan*, klik *Cancel* jika ingin membatalkan
