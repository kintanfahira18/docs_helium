# Spring4Shell
Deteksi dan mitigasi kerentanan *zero-day* baru di *Java Spring Framework*.

### Akses *Spring4Shell* melalui Menu Navigasi
:::note Catatan

Pengguna dapat melakukan pemindaian menggunakan VAPT Tools melalui menu navigasi "VAPT Tools". Dengan mengklik menu "VAPT Tools", akan muncul berbagai macam alat, kemudian klik alat *Spring4Shell*.

:::
![Select Spring4Shell Navbar Capture](/img/capture/vapt_navbar/Select-spring4shell-navbar.png)

![Spring4Shell Capture](/img/capture/vapt_navbar/spring4shell.png)

Untuk *scanning* menggunakan *Spring4Shell*, pengguna dapat mengikuti langkah-langkah berikut.
1. Inputkan nama *task* pada *textbox Task Name* 
2. Inputkan nilai target berupa *domain/URL/IPs* 
3. Klik *Scheduled* untuk *Enabled* atau aktifkan *Schedule Scan* (opsional)
4. Klik *Multiples Scans* untuk *Enabled* atau aktifkan *Multiple Scans* (opsional)
5. Klik *checkbox agree Terms of Service*
6. Klik tombol ***Apply*** untuk memulai *scan*

### Akses *Spring4Shell* melalui Halaman Targets
:::note Catatan

Target yang dipilih akan dipindai dengan alat yang tersedia. Pilih Alat *Spring4Shell*.

:::
![Select Spring4Shell Capture](/img/capture/vapt_target/Select-spring4shell.png)

![Spring4Shell Capture](/img/capture/vapt_target/spring4shell.png)

Untuk *scanning* menggunakan *Spring4Shell*, pengguna dapat mengikuti langkah-langkah berikut.
1. Inputkan nama *task* pada *textbox Task Name*  
2. Klik *Enabled* untuk mengaktifkan *Multiple Scan* (opsional)
3. Klik *Enabled* untuk mengaktifkan *Schedule Scan* (opsional)
4. Klik *checkbox agree Terms of Service*
5. Klik tombol *Start Scan* untuk memulai *scan*, klik *Cancel* jika ingin membatalkan
