# XSS Scanner
Menemukan kerentanan *Cross-Site Scripting (XSS)* dalam aplikasi web.

### Akses *XSS Scanner* melalui Menu Navigasi
:::note Catatan

Pengguna dapat melakukan pemindaian menggunakan VAPT Tools melalui menu navigasi "VAPT Tools". Dengan mengklik menu "VAPT Tools", akan muncul berbagai macam alat, kemudian klik alat *XSS Scanner*.

:::
![Select XSS Scanner Navbar Capture](/img/capture/vapt_navbar/Select-XSS-scanner-navbar.png)

![XSS Scanner Capture](/img/capture/vapt_navbar/XSS-scanner.png)

Untuk *scanning* menggunakan *XSS Scanner*, pengguna dapat mengikuti langkah-langkah berikut.
1. Inputkan nama *task* pada *textbox Task Name* 
2. Inputkan nilai target berupa *domain/URL/IPs* 
3. Klik *Scheduled* untuk *Enabled* atau aktifkan *Schedule Scan* (opsional)
4. Klik *Multiples Scans* untuk *Enabled* atau aktifkan *Multiple Scans* (opsional)
5. Klik *checkbox agree Terms of Service*
6. Klik tombol ***Apply*** untuk memulai *scan*

### Akses *XSS Scanner* melalui Halaman Targets
:::note Catatan

Target yang dipilih akan dipindai dengan alat yang tersedia. Pilih Alat *XSS Scanner*.

:::
![Select XSS Scanner Capture](/img/capture/vapt_target/Select-XSS-scanner.png)

![XSS Scanner Capture](/img/capture/vapt_target/XSS-scanner.png)

Untuk *scanning* menggunakan *XSS Scanner*, pengguna dapat mengikuti langkah-langkah berikut.
1. Inputkan nama *task* pada *textbox Task Name*  
2. Klik *Enabled* untuk mengaktifkan *Multiple Scan* (opsional)
3. Klik *Enabled* untuk mengaktifkan *Schedule Scan* (opsional)
4. Klik *checkbox agree Terms of Service*
5. Klik tombol *Start Scan* untuk memulai *scan*, klik *Cancel* jika ingin membatalkan
