# CMS Scanner
Mendeteksi kelemahan keamanan pada *CMS* paling populer.

### Akses *CMS Scanner* melalui Menu Navigasi
:::note Catatan

Pengguna dapat melakukan pemindaian menggunakan VAPT Tools melalui menu navigasi "VAPT Tools". Dengan mengklik menu "VAPT Tools", akan muncul berbagai macam alat, kemudian klik alat *CMS Scanner*.

:::
![Select CMS Scanner Navbar Capture](/img/capture/vapt_navbar/Select-CMS-scanner-navbar.png)

![CMS Scanner Capture](/img/capture/vapt_navbar/CMS-scanner.png)

Untuk *scanning* menggunakan *CMS Scanner*, pengguna dapat mengikuti langkah-langkah berikut.
1. Inputkan nama *task* pada *textbox Task Name* 
2. Inputkan nilai target berupa *domain/URL/IPs* 
3. Pilih salah satu *scan option* yaitu antara *Full Scan* atau *Basic Scan*
4. Pilih salah satu *CMS Detection*
5. Klik *Scheduled* untuk *Enabled* atau aktifkan *Schedule Scan* (opsional)
6. Klik *Multiples Scans* untuk *Enabled* atau aktifkan *Multiple Scans* (opsional)
7. Klik *checkbox agree Terms of Service*
8. Klik tombol ***Apply*** untuk memulai *scan*

### Akses *CMS Scanner* melalui Halaman Targets
:::note Catatan

Target yang dipilih akan dipindai dengan alat yang tersedia. Pilih Alat *CMS Scanner*

:::
![Select CMS Scanner Capture](/img/capture/vapt_target/Select-CMS-scanner.png)

![CMS Scanner Capture](/img/capture/vapt_target/CMS-scanner.png)

Untuk *scanning* menggunakan *CMS Scanner*, pengguna dapat mengikuti langkah-langkah berikut.
1. Inputkan nama *task* pada *textbox Task Name*  
2. Klik *Enabled* untuk mengaktifkan *Multiple Scan* (opsional)
3. Pilih salah satu *scan option* yaitu antara *Full Scan* atau *Basic Scan*
4. Pilih salah satu *CMS Detection* 
5. Klik *Enabled* untuk mengaktifkan *Schedule Scan* (opsional)
6. Klik *checkbox agree Terms of Service*
7. Klik tombol *Start Scan* untuk memulai *scan*, klik *Cancel* jika ingin membatalkan
