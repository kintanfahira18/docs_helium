# Find Git Exposure
Menemukan kode sumber melalui folder .git yang terbuka.

### Akses *Find Git Exposure* melalui Menu Navigasi
:::note Catatan

Pengguna dapat melakukan pemindaian menggunakan VAPT Tools melalui menu navigasi "VAPT Tools". Dengan mengklik menu "VAPT Tools", akan muncul berbagai macam alat, kemudian klik alat *Find Git Exposure*.

:::
![Select Find Git Exposure Navbar Capture](/img/capture/vapt_navbar/Select-find-git-exposure-navbar.png)

![Find Git Exposure Capture](/img/capture/vapt_navbar/find-git-exposure.png)

Untuk *scanning* menggunakan *Find Git Exposure*, pengguna dapat mengikuti langkah-langkah berikut.
1. Inputkan nama *task* pada *textbox Task Name* 
2. Inputkan nilai target berupa *domain/URL/IPs* 
3. Klik *Scheduled* untuk *Enabled* atau aktifkan *Schedule Scan* (opsional)
4. Klik *Multiples Scans* untuk *Enabled* atau aktifkan *Multiple Scans* (opsional)
5. Klik *checkbox agree Terms of Service*
6. Klik tombol ***Apply*** untuk memulai *scan*

### Akses *Find Git Exposure* melalui Halaman Targets
:::note Catatan

Target yang dipilih akan dipindai dengan alat yang tersedia. Pilih Alat *Find Git Exposure*.

:::
![Select Find Git Exposure Capture](/img/capture/vapt_target/Select-find-git-exposure.png)

![Find Git Exposure Capture](/img/capture/vapt_target/find-git-exposure.png)

Untuk *scanning* menggunakan *Find Git Exposure*, pengguna dapat mengikuti langkah-langkah berikut.
1. Inputkan nama *task* pada *textbox Task Name*  
2. Klik *Enabled* untuk mengaktifkan *Multiple Scan* (opsional)
3. Klik *Enabled* untuk mengaktifkan *Schedule Scan* (opsional)
4. Klik *checkbox agree Terms of Service*
5. Klik tombol *Start Scan* untuk memulai *scan*, klik *Cancel* jika ingin membatalkan
