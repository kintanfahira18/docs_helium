# Google Hacking
Teknik peretas yang memanfaatkan Google Penelusuran dan berbagai aplikasi Google lainnya untuk mencari kerentanan pada konfigurasi dan kode komputer yang digunakan oleh situs web. Dengan cara melakukan *dorking* pada target sesuai dengan tipe dork yang dipilih pengguna.

### Akses *Google Hacking* melalui Menu Navigasi
:::note Catatan

Pengguna dapat melakukan pemindaian menggunakan VAPT Tools melalui menu navigasi "VAPT Tools". Dengan mengklik menu "VAPT Tools", akan muncul berbagai macam alat, kemudian klik alat *Google Hacking*.

:::
![Select Google Hacking Navbar Capture](/img/capture/vapt_navbar/Select-google-hacking-navbar.png)

![Google Hacking Capture](/img/capture/vapt_navbar/google-hacking.png)

Untuk *scanning* menggunakan *Google Hacking*, pengguna dapat mengikuti langkah-langkah berikut.
  ![Google Hacking Capture](/img/capture/vapt_navbar/google-hacking.png)
1. Inputkan nilai target berupa *domain/URL/IPs* 
2. Klik *Select Google Dorks*, maka akan muncul pilihan *google dorks*
   ![Google Dorks Capture](/img/capture/vapt_navbar/google-dorks.png)
3. Pilih salah satu *google dorks*
4. Klik *checkbox agree Terms of Service*
5. Klik tombol ***Apply*** untuk memulai *scan*

:::info 

Untuk menggunakan *tools Google Hacking* pengguna hanya dapat mengakses melalui menu navigasi.

:::
