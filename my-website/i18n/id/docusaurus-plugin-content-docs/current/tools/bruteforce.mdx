# Bruteforce
Menemukan kredensial layanan di Aplikasi Web, *SSH*, *FTP*, *MySQL*, *Telnet*, dan banyak lagi.

### Akses *Bruteforce* melalui Menu Navigasi
:::note Catatan 


Pengguna dapat melakukan pemindaian menggunakan VAPT Tools melalui menu navigasi "VAPT Tools". Dengan mengklik menu "VAPT Tools", akan muncul berbagai macam alat, kemudian klik alat *Bruteforce*.

:::
![Select Bruteforce Navbar Capture](/img/capture/vapt_navbar/Select-bruteforce-navbar.png)

![Bruteforce Capture](/img/capture/vapt_navbar/bruteforce.png)

Untuk *scanning* menggunakan *Bruteforce*, pengguna dapat mengikuti langkah-langkah berikut.
1. Inputkan nama *task* pada *textbox Task Name* 
2. Inputkan nilai target berupa *domain/URL/IPs* 
3. Pilih salah satu *service* 
4. Pilih *Wordlist Username* 
5. Pilih *Wordlist Password*
6. Klik *Scheduled* untuk *Enabled* atau aktifkan *Schedule Scan* (opsional)
7. Klik *Multiples Scans* untuk *Enabled* atau aktifkan *Multiple Scans* (opsional)
8. Klik *checkbox agree Terms of Service*
9. Klik tombol ***Apply*** untuk memulai *scan*

### Akses *Bruteforce* melalui Halaman Targets
:::note Catatan

Target yang dipilih akan dipindai dengan alat yang tersedia. Pilih Alat *Bruteforce*

:::
![Select Bruteforce Capture](/img/capture/vapt_target/Select-bruteforce.png)

![Bruteforce Capture](/img/capture/vapt_target/bruteforce.png)

Untuk *scanning* menggunakan *Bruteforce*, pengguna dapat mengikuti langkah-langkah berikut.
1. Inputkan nama *task* pada *textbox Task Name*  
2. Pilih salah satu *service* 
3. Pilih *Wordlist Username* 
4. Pilih *Wordlist Password*
5. Klik *Enabled* untuk mengaktifkan *Schedule Scan* (opsional)
6. Klik *checkbox agree Terms of Service*
7. Klik tombol *Start Scan* untuk memulai *scan*, klik *Cancel* jika ingin membatalkan
