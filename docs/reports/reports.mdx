# Reports
The Reports page contains a list of reports from targets that have been scanned.
![Reports Capture](/img/capture/reports.png)

:::info

- **Task Name** is the name listed for the target
- **Tool Name** is the tool used when the target is scanned
- **Target** is the target to be scanned
- Scan date is the date and time when the scan was initiated
- **Vulnerability** are rated by severity with colors: red for high, yellow for medium, blue for low, and green for informational. However, this vulnerability feature appears only in the VA Scanner tool
- **Action** is a view button to view the scan report

:::

## View Scan Reports

:::note

To view or export a scan report, users can click the View button, then they will be directed to the scan report document.

:::
