# CMS Scanner
Detecting security flaws of the foremost popular CMSs.

### Access CMS Scanner via the Navigation Menu
:::note

Users can scan using VAPT Tools via the "VAPT Tools" navigation menu. By clicking on the "VAPT Tools" menu, various tools will appear, then click CMS Scanner tool.

:::
![Select CMS Scanner Navbar Capture](/img/capture/vapt_navbar/Select-CMS-scanner-navbar.png)

![CMS Scanner Capture](/img/capture/vapt_navbar/CMS-scanner.png)

To scan using the CMS Scanner, users can follow the steps below.
1. Enter the task name in the Task Name textbox 
2. Input the target value in the form of domain/URL/IPs 
3. Select one of the scan options, namely Full Scan or Basic Scan 
4. Select one of the CMS Detection 
5. Click Scheduled to Enabled schedule scan(optional)
6. Click Multiples Scans to Enabled Multiple Scans (optional)
7. Click the agree Terms of Service checkbox 
8. Click the **Apply** button to start the scan

### Access CMS Scanner Via Targets Page
:::note

The selected target will be scanned with the available tools. Select CMS Scanner Tool

:::
![Select CMS Scanner Capture](/img/capture/vapt_target/Select-CMS-scanner.png)

![CMS Scanner Capture](/img/capture/vapt_target/CMS-scanner.png)

To scan using the CMS Scanner, users can follow the steps below.
1. Enter the task name in the Task Name textbox
2. Click Enabled to enable Multiple Scan (optional)
3. Select one of the scan options, namely Full Scan or Basic Scan
4. Select one of the CMS Detection
5. Click Enabled to activate Schedule Scan (optional)
6. Click the agree Terms of Service checkbox
7. Click the **Start Scan** button to start the scan, click Cancel if you want to cancel
