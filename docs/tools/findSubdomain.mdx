# Find Subdomain
Discover subdomains and enumerate the vulnerabilities.

### Access Find Subdomain via the Navigation Menu
:::note

Users can scan using VAPT Tools via the "VAPT Tools" navigation menu. By clicking on the "VAPT Tools" menu, various tools will appear, then click Find Subdomain tool.

:::
![Select Find Subdomain Navbar Capture](/img/capture/vapt_navbar/Select-find-subdomain-navbar.png)

![Find Subdomain Capture](/img/capture/vapt_navbar/find-subdomain.png)

To scan using Find Subdomain, users can follow these steps.
1. Enter the task name in the Task Name textbox 
2. Input the target value in the form of domain/URL/IPs 
3. Click Scheduled to Enabled Schedule Scan (optional)
4. Click Multiples Scans to Enabled Multiple Scans (optional)
5. Click the agree Terms of Service checkbox 
6. Click the **Apply** button to start the scan

### Access Find Subdomain Via Targets Page
:::note

The selected target will be scanned with the available tools. Select Find Subdomain Tool

:::
![Select Find Subdomain Capture](/img/capture/vapt_target/Select-find-subdomain.png)

![Find Subdomain Capture](/img/capture/vapt_target/find-subdomain.png)

To scan using Find Subdomain, users can follow these steps.
1. Enter the task name in the Task Name textbox 
2. Click Enabled to enable Multiple Scan (optional)
3. Click Enabled to activate Schedule Scan (optional)
4. Click the agree Terms of Service checkbox 
5. Click the **Start Scan** button to start the scan, click Cancel if you want to cancel
