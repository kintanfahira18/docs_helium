# Find Git Exposure
Discover source code via the exposed .git folder.

### Access Find Git Exposure via the Navigation Menu
:::note

Users can scan using VAPT Tools via the "VAPT Tools" navigation menu. By clicking on the "VAPT Tools" menu, various tools will appear, then click Find Git Exposure tool.

:::
![Select Find Git Exposure Navbar Capture](/img/capture/vapt_navbar/Select-find-git-exposure-navbar.png)

![Find Git Exposure Capture](/img/capture/vapt_navbar/find-git-exposure.png)

To scan using Find Git Exposure, users can follow these steps.
1. Enter the task name in the Task Name textbox 
2. Input the target value in the form of domain/URL/IPs 
3. Click Scheduled to Enabled Schedule Scan (optional)
4. Click Multiples Scans to Enabled Multiple Scans (optional)
5. Click the agree Terms of Service checkbox 
6. Click the **Apply** button to start the scan

### Access Find Git Exposure Via Targets Page
:::note

The selected target will be scanned with the available tools. Select Find Git Exposure Tool

:::
![Select Find Git Exposure Capture](/img/capture/vapt_target/Select-find-git-exposure.png)

![Find Git Exposure Capture](/img/capture/vapt_target/find-git-exposure.png)

To scan using Find Git Exposure, users can follow these steps.
1. Enter the task name in the Task Name textbox 
2. Click Enabled to enable Multiple Scan (optional)
3. Click Enabled to activate Schedule Scan (optional)
4. Click the agree Terms of Service checkbox 
5. Click the **Start Scan** button to start the scan, click Cancel if you want to cancel
